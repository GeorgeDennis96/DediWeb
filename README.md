DeDi's Death Dice

## SUMMARY

DeDi's death dice is a discord interaction game where players roll a die to determine their fate. 
The game is simple: each player takes turns rolling the initial number or the new number that has been rolled.
Loser is whoever hits the number 1.

## COMMANDS
- `/dediregister`
- `/dedi [user] [wager]`
- `/dedidaily`
- `/dediwho [user]`
- `/dedirules`
- `/deditrade`

## FEATURES
- Simple and fun gameplay.
- Dashboard to View player statistics.
- Players can earn points by winning duels.
- Slash command `/DediDaily` to claim daily reward of 10000 points.
- Players can choose their own wager to make the game more exciting.
- The wager is the initial number you start the game with. You can choose any number you want, but it must be greater than 1 and you must have the balance to initiate the duel.
- A leaderboard/ranking system to track the top players and their points. **(Work In Progress)**
